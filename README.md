# About

This open-source project gathers small signals from the world and turns them into daily broadcasts of art, language, and atmosphere.

Using public data, curated artwork, short written messages, and generative AI, it tries to shape something clear and gentle—something that can inform, but also quietly accompany a moment in the day. The art is curated by friends and family, so the project grows not only from data, but also from taste, memory, and shared sensibility.

The goal is to keep each broadcast simple, beautiful, and easy to receive: a small composition made from information, care, and imagination.
